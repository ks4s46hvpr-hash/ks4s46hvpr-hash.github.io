
## 页面说明

### 首页 (index.html)
- **功能**：展示公司六大服务承诺
- **特色**：响应式卡片布局、滚动动画、联系客服板块、微信公众号二维码区域
- **布局**：承诺展示 → 企业信念 → 联系客服 → 微信公众号

### 关于我们 (about.html)
- **功能**：公司详细介绍和企业文化展示
- **内容**：公司简介、资质证书、经营理念、质量方针、企业信念、社会承诺
- **特色**：时间线式企业历程、价值观卡片、承诺列表

### 服务介绍 (services.html)
- **功能**：认证项目分类展示
- **分类**：管理体系认证、服务认证、其他认证
- **特色**：清晰的分类展示、详细的项目列表

### 联系我们 (contact.html)
- **功能**：全面的联系方式信息
- **内容**：公司总机、合同受理、审核安排、培训联系、市场客服、监督电话
- **特色**：分类清晰的联系信息、公司网址、QQ客服

## 技术特点

### 前端技术
- **HTML5**：语义化标签，良好的SEO结构
- **CSS3**：现代布局技术（Grid + Flexbox）、CSS变量、动画效果
- **JavaScript**：原生JS实现交互功能，Anime.js动画库
- **响应式设计**：适配手机、平板、桌面设备

### 核心功能
- 滚动触发动画效果
- 卡片悬停交互
- 平滑滚动导航
- 响应式布局适配
- 微信公众号二维码展示区域

### 第三方资源
- Google Fonts：Noto Serif SC 和 Noto Sans SC 字体
- Anime.js：流畅的动画效果
- CDN引入的图标和脚本资源

## 使用说明

### 部署环境
- 支持任何标准的Web服务器（Apache、Nginx等）
- 无需后端支持，纯静态网站
- 建议启用HTTPS协议

### 文件路径
- 所有页面文件可直接在根目录访问
- 外部资源使用CDN引入，确保网络连接
- 本地图片资源建议放在assets/images/目录

### 自定义修改
1. **公司Logo**：替换header中的logo.svg
2. **联系方式**：修改各页面的联系电话和QQ号
3. **微信公众号二维码**：替换wechat-qrcode区域的占位图片
4. **颜色主题**：修改CSS变量中的颜色值
5. **内容更新**：直接在HTML文件中修改文本内容

## 性能优化建议

1. **图片优化**：
   - 使用WebP格式图片
   - 压缩图片文件大小
   - 添加图片懒加载

2. **代码优化**：
   - 合并和压缩CSS/JS文件
   - 使用CDN加速静态资源
   - 启用Gzip压缩

3. **SEO优化**：
   - 添加meta描述标签
   - 优化页面标题
   - 使用语义化HTML标签

## 安全建议

1. 定期更新第三方库版本
2. 对用户输入进行验证（如有表单）
3. 使用HTTPS协议
4. 设置适当的安全响应头
